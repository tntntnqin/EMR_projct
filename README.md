# EMR_projct

### 프로젝트 소개
환자에 대한 진료기록을 컴퓨터를 이용해 의무기록을 전자화한 시스템.
의료진이 직접 프로그램을 통해 데이터를 입력하고, 입력된 데이터를 데이터베이스로 관리하여 업무의 효율을 높인다.

### 팀원
***
* 강민경
* 서예선
* 오수빈
* 정진해
